## Instalar Terminal

- Windows Terminal: Multiples terminales

```bash
  winget install --id=Microsoft.WindowsTerminal  -e
```

- PowerShell : Terminal actualizada de windows

```bash
  winget install --id=Microsoft.PowerShell  -e
```
